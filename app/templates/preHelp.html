Preprocessing Help

General

The preprocessing step uses statistics and natural language processing techniques to reduce spell errors and different word forms. This is realized through a similarity replacement algorithm which computes how similar two words are - and replaces the unimportant with important ones. Importance is defined by the formula you already provided. The threshold defines how many word will be replaced.

Information

On top is a table with two lines, current and new. Current is a automatically created default and a good starting point. The threshold means how similar two words need to be replaced - 0.65: 65% similarity. Replacements shows how many words will be replaced with the given threshold.

Histogram

The histogram below the list shows the number of replacements binned over similarity values - 10 at the 0.1-0.2 bin means that there are 10 replacements within the range 0.1 and 0.2. You can zoom in by dragging a window or clicking onto a bin. Zooming updates the range of the slider below.

Slider

The slider is used to find a better threshold by simple dragging it around. If you change the slider, the grid below updates accoring to the new threshold.

Grid

The grid shows all replacements around the selected threshold. The selected one is highlighted. The number of entries varies with the number of replacements around the threshold. In the right top corner is a dropdown menu where you may export the information in the grid.

Symbols

Back: Reverts one zooming step per click.
Overview: Returns to the inital overview.
Advanced: Shows more information about this step.
